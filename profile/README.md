## Installation Instructions

1. Download the zip from above link.  
2. Extract all files into a folder (e.g., `C:/AnchorSolana`).  
3. Ensure Rust and Solana CLI tools are installed on your PC.  
4. Open a terminal and navigate to the folder.  
5. Run `anchor build` to compile your Solana programs.  
6. Deploy your program with `anchor deploy` and start building decentralized apps!
